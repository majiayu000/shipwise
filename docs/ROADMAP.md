# Roadmap

## v0.2: Documentation-first guide

Status: current.

- Generic README.
- Decision tree.
- Release lifecycle checklist.
- Distribution template.
- Template system.
- Success metrics.
- Feedback loop.
- Project scaffold script.

## v0.3: First validated launch

Requirement: run one real project through the full system.

- Fill a complete playbook for `ccstats`, `remem`, or another real project.
- Add real proof, screenshots, and metrics.
- Complete Day 1 and Day 7 review.
- Update at least one template based on evidence.

## v0.5: Light documentation automation

Only after v0.3 is complete.

- Validate project launch folder completeness.
- Generate draft documents from templates.
- Check broken links and missing source references.
- Produce a launch readiness report.

## v1.0: Agent-readable launch planner

Only after several launches prove stable patterns.

- Interactive project classification.
- Platform source map and recommendation logic.
- Template filling workflow for docs and drafts.
- Metrics import into project markdown.
- Optional memory/backend integrations.

## Non-goals

- No auto-posting as part of this repository.
- No credentials in this repo.
- No fake metrics.
- No platform-specific growth hacks that violate community rules.
