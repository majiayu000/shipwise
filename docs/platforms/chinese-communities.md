# Chinese Communities Source Guide

Chinese developer channels are useful for localized adoption, explanation, and feedback. Treat each as a separate community, not a repost target.

## Common Channels

- WeChat official account.
- Juejin.
- V2EX.
- Zhihu.
- Bilibili.
- OSChina.
- Jike.
- Xiaohongshu when visuals/story fit.

## Source Rule

Each platform has its own posting, moderation, and account rules. Before publishing, check the current platform rules and the specific community norms.

## Use When

- The target users include Chinese developers.
- Installation and docs can be explained in Chinese.
- Screenshots or demos help reduce friction.
- There is a practical story, not just an announcement.

## Agent Checklist

- [ ] Localize the pain point and examples.
- [ ] Include install and quickstart commands.
- [ ] Use screenshots or GIFs when helpful.
- [ ] Avoid inflated claims.
- [ ] Provide a feedback path.
- [ ] Record platform links and comments.

## Do Not

- Do not treat all Chinese platforms as one audience.
- Do not paste an English launch post through translation only.
- Do not use fake numbers, exaggerated savings, or unsupported "best/first/largest" claims.
