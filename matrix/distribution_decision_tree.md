# Distribution Decision Matrix

| Archetype | Must have | First wave | Second wave | Avoid first |
|---|---|---|---|---|
| CLI/tool | release artifact, install command, terminal demo | GitHub, package registry, X, HN/Reddit | blog, Chinese community, newsletter | Product Hunt without visuals |
| Library/SDK | package, docs, API example | registry, GitHub, example repo, blog | X, HN, newsletter | broad social without docs |
| AI workflow | integration proof, demo video, limitation list | GitHub, X, niche Reddit/Discord | HN, Chinese video/article | claims without workflow demo |
| Framework/guardrail | config examples, before/after, false-positive policy | GitHub, technical article, X | HN, Reddit | moralizing quality claims |
| Registry/dataset | schema, sample query, coverage stats | GitHub, HN, X | blog, newsletter | "largest" claims without data |
| App/UI | live demo, screenshots, onboarding | Product Hunt, X, LinkedIn | blog, video, communities | HN Show HN if not technically interesting |
| Writeup | data, method, conclusion | blog, HN regular post, Reddit | newsletter, X | Show HN |
